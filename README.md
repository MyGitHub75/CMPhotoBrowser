# 系统支持

* iOS7+

* Xcode7

## 安装 

### CocoaPods

* 在 Podfile 中输入以下内容：
```
pod 'CMPhotoBrowser'
```
### 其他方法
* 下载文件包，将包（CMPhotoBrowser）拖入程序
* 

```
#import "CMPhotoBrowser.h"
#import "CMPhoto.h"
```
